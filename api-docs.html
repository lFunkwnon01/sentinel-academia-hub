Sentinel AcademIA API (0.1.0)

Download OpenAPI specification:

Equipo Sentinel AcademIA: sentinel-academia@universidad.edu License: MIT

API REST para la plataforma Sentinel AcademIA - gestion de quejas universitarias con analisis automatico mediante IA generativa (Oracle OCI Cohere Command R + Llama 3.1).

Caracteristicas

  • Endpoints asincronos (202 Accepted) para operaciones de escritura
  • Validacion estricta con Zod en backend (mismo schema que en este spec)
  • Correlation ID end-to-end (header x-correlation-id)
  • Rate limiting (100 req/min por usuario, 200 por IP)
  • Respuestas de error uniformes con code, message, correlationId

quejas

Operaciones CRUD de quejas

Crear nueva queja

Recibe una queja y la encola para procesamiento asincrono. Devuelve 202 Accepted con un ID de seguimiento inmediatamente. El analisis con LLM se completa en background (< 60s tipicamente).

Request Body schema: application/json
required
titulo
required
string [ 5 .. 120 ] characters

Titulo breve de la queja

descripcion
required
string [ 20 .. 5000 ] characters

Descripcion detallada del problema

categoriaDeclarada
required
string
Enum: "ACADEMICA" "INFRAESTRUCTURA" "ACOSO" "ADMINISTRATIVA" "SALUD" "OTRA"

Categoria que el reportante considera adecuada

adjuntos
Array of strings <uri> <= 5 items [ items <uri > ]

URLs a archivos adjuntos (PDFs, imagenes) en S3

anonima
boolean
Default: false

Si el reporte es anonimo

sede
string <= 100 characters
facultad
string <= 100 characters
contactoEmail
string <email>

Email de contacto (solo si no es anonimo)

contactoTelefono
string^\+?[0-9\s-]{7,20}$
